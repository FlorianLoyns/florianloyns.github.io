Zu den Trainings → Werkzeuge für Lehrende

Warum gibt es das hier? Ich unterrichte als Lehrer für Pflegeberufe in der generalistischen Pflegeausbildung. Diese Werkzeuge sind aus dem Unterrichtsalltag entstanden und bleiben offen – weil gute Didaktik niemandem gehören muss.

Was hier landet, hat mindestens eine Klasse oder eine Kollegen-Diskussion hinter sich. Ich bin kein Programmierer, sondern Pädagoge: didaktisch gedacht, technisch pragmatisch. Alles läuft direkt im Browser ohne Anmeldung, ohne Tracking, ohne externe Server — auch offline auf jedem Schulrechner und Smartphone.

Die Tests sind Reflexionsimpulse, keine Diagnostik. Die Werkzeuge ersetzen keine Pädagogik, sie beschleunigen sie. Feedback, Fehlerhinweise und Vorschläge sind willkommen – gern per Mail oder als Issue auf GitHub.

So installierst du die Apps auf dem Handy

Tools mit dem PWA-Badge lassen sich auf jedem Smartphone wie eine App installieren — direkt aus dem Browser, ohne App Store, ohne Anmeldung. Einmal eingerichtet, läuft die App auch offline.

iPhone & iPad · Safari

Auf iOS installieren

  1. Tool im Safari-Browser öffnen
  2. Auf das Teilen-Symbol unten tippen (Quadrat mit Pfeil nach oben)
  3. Nach unten scrollen, „Zum Home-Bildschirm" wählen
  4. Namen bestätigen — fertig
Android · Chrome

Auf Android installieren

  1. Tool im Chrome-Browser öffnen
  2. Auf das Drei-Punkte-Menü oben rechts tippen
  3. „App installieren" oder „Zum Startbildschirm hinzufügen" wählen
  4. Bestätigen — fertig
